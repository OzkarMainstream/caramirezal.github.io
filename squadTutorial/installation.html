3. Installation

In theory, SQUAD installation can be done easily by installing and loading "devtools" R package and then using the install_github command in R. However, it is strongly suggested to install dependencies prior to installation and checking they are working properly. SQUAD has the following dependencies: rJava, BoolNet, deSolve and reShape2. Here, I provide some notes for installing dependencies.

BoolNet and devtools installation may require prior instalation of libcurl and xml packages. Please, check if they are currently installed. On linux system you can execute the following command in the system shell to install both packages, as detailed here.

> sudo apt-get install libcurl4-openssl-dev libxml2-dev
> sudo apt-get install libcurl4-openssl-dev libssl-dev
> sudo apt-get update

Install devtools library by running the following command on R:

> install.packages("devtools").

Correct rJava installation is critical to install the package. Installing rJava can be tricky, you may need to update your Java version system and then run the following commands on shell as suggested here.

> /usr/lib/jvm/java-8-oracle/jre/lib/amd64
> /usr/lib/jvm/java-8-oracle/jre/lib/amd64/server
> sudo ldconfig
> sudo apt-get install r-cran-rjava

Then, restart Rstudio and install rJava.

WINDOWS NOTES: In windows systems rJava installation may require to follow the steps suggested here . Download the latest version of Java available here, install it in your system.

Finally, install the SQUAD package from the repository using the next command on R:

> library(devtools)
> install_github("caramirezal/SQUAD/SQUAD")

Finally, you can load the library using the standard R form:

> library(SQUAD)
